Project 01 · Architecture Map

Building the Creator Ops Flow

TEMIS는 일정표 제작 도구, 템플릿 상점, 주문 파일, 권한/정산 데이터를 하나의 운영 흐름으로 연결한 크리에이터 커머스 플랫폼입니다.

Next.js TypeScript TanStack Query Supabase Cloudflare R2 Nodemailer
CREATOR STUDIO

일정표 편집기

입력 폼, 프리뷰, 저장/복원 로직을 설정 기반 hook으로 연결.

Editor State

템플릿마다 다른 입력 필드는 설정으로 주입하고, 공통 편집 로직은 재사용합니다.

TEMPLATE SHOP

상점/구매 요청

템플릿 탐색, 구매 요청, 접근 권한 확인이 이어지는 상점 흐름.

Commerce Flow

React Query와 service layer로 목록, 상세, mutation, invalidate를 분리합니다.

ORDER DESK

커스텀 주문 운영

구매 요청, 주문 상태, 관리자 작업을 API Route에서 처리.

Next.js API Routes

인증, 권한, 구매 요청, 관리자 액션을 얇은 서버 계층으로 정리합니다.

FILE VAULT

Cloudflare R2

주문 파일과 이미지를 검증하고 R2 key와 DB record를 연결.

Upload Pipeline

업로드, 삭제, zip 다운로드 흐름을 분리해 실패 추적과 운영 관리를 쉽게 만듭니다.

OPS CORE

Supabase DB

사용자, 템플릿, 구매 요청, 접근 권한, 판매/정산 데이터를 분리.

Database Relations

제작 도구보다 먼저 운영 데이터 관계를 고정해 권한과 정산 흐름을 안정화합니다.

EMAIL TOKENS

Nodemailer

초대, 인증, 비밀번호 재설정 토큰 메일을 서비스 흐름에 연결.

ADMIN OPS

관리자/정산 대시보드

템플릿, 주문, 파일, 접근 권한, 정산 상태를 한 화면에서 관리.

Operational Loop

제작 → 판매 → 파일 전달 → 권한 부여 → 정산 확인까지 하나의 제품 경험으로 닫힙니다.
